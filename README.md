# Rugby Run

A 2D endless runner built with HTML5 Canvas and vanilla JavaScript. Dodge incoming rugby players and flying balls to survive as long as possible.

## How to play

Open `project/menu.html` in a browser (use a local server like VS Code Live Server to avoid CORS issues with assets).

| Key | Action |
|-----|--------|
| ↑ | Jump |
| ↓ (hold) | Slide |
| ESC | Back to menu |

## Levels

- **Level 1** — Jump only (rugby players)
- **Level 2** — Slide only (balls)
- **Level 3** — Both

## Speed

Choose the difficulty before starting:

| Speed | Opponent |
|-------|----------|
| 1 | Castres |
| 2 | La Rochelle |
| 3 | Bordeaux |
